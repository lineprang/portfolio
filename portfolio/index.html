Webapp project - PodCut

Description
Today there are a huge variety of podcasts to choose between. It can be diffucult to navigate through. PodCut is the solution. PodCut is a podcast guide, that generates podcast ideas that matches the users intersts and streaming services.

Purpose
PodCut is our latest school project and the purpose of this project was to make a webapp showing our skills using Single Page Application structure, as well as using APIs and fetching and storing data.

Resposibility
Idea generation, research, testing, UX/UI, programming
